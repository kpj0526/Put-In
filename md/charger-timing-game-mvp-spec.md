# 충전기 타이밍 게임 MVP 기능명세서

## 1. 게임 개요

플레이어가 좌우로 움직이는 충전기를 타이밍에 맞춰 멈춰서 콘센트에 정확히 꽂는 캐주얼 타이밍 게임이다.

목표는 정확도 100%에 가깝게 충전기를 꽂고, 로그인된 계정으로 리더보드에서 높은 순위를 기록하는 것이다.

## 2. 핵심 콘셉트

- 한 판이 짧다.
- 조작은 클릭, 터치, 키보드 입력 한 번이다.
- 결과는 정확도 퍼센트로 나온다.
- 플레이어는 이메일 계정으로 로그인하고 더 높은 정확도와 리더보드 순위를 위해 반복 도전한다.

## 3. 기본 게임 흐름

1. 회원가입 또는 로그인을 한다.
2. 시작 버튼을 누른다.
3. 충전기가 좌우로 움직인다.
4. 플레이어가 원하는 타이밍에 화면을 누르거나 키보드를 입력한다.
5. 충전기가 멈춘다.
6. 콘센트 중심과 충전기 중심의 거리로 정확도를 계산한다.
7. 정확도와 판정 문구를 보여준다.
8. 로그인된 계정의 닉네임으로 기록을 리더보드에 등록한다.
9. 리더보드 순위를 보여준다.
10. 다시하기 버튼으로 재도전한다.

## 4. 조작 방식

### PC

- 마우스 클릭
- 스페이스바
- 엔터키

### 모바일

- 화면 터치

입력은 한 판에 한 번만 가능하다.

## 5. 주요 오브젝트

### 충전기

#### 기능

- 화면에서 좌우로 자동 이동한다.
- 일정 속도로 움직인다.
- 화면 끝에 닿으면 방향을 반대로 바꾼다.
- 플레이어가 입력하면 즉시 멈춘다.

### 콘센트

#### 기능

- 화면 중앙에 고정되어 있다.
- 충전기가 꽂혀야 하는 목표 위치이다.
- MVP에서는 위치가 변하지 않는다.

### 배터리 UI

#### 기능

- 시작 시 배터리 1% 상태로 표시한다.
- 정확도 결과에 따라 충전량을 표시한다.

#### 예시

- 정확도 87% -> 배터리 87% 충전

### 리더보드

#### 기능

- 플레이 결과를 정확도 기준으로 순위화한다.
- 정확도가 높은 기록이 상위에 노출된다.
- 동일 정확도일 경우 먼저 등록된 기록을 우선한다.
- MVP에서는 상위 10개 기록만 표시한다.

#### 기록 항목

- 순위
- 유저 ID
- 닉네임
- 정확도
- 판정 문구
- 기록 시간

### 회원 계정

#### 기능

- 이메일과 비밀번호로 회원가입한다.
- 이메일과 비밀번호로 로그인한다.
- 로그인 성공 시 JWT access token과 refresh token을 발급한다.
- refresh token으로 access token을 재발급한다.
- 로그아웃 시 refresh token을 폐기한다.

#### 입력 조건

- 이메일은 올바른 이메일 형식이어야 한다.
- 비밀번호는 8~64자여야 한다.
- 비밀번호는 영문 대문자, 영문 소문자, 숫자, 특수문자를 각각 1개 이상 포함해야 한다.
- 닉네임은 2~12자여야 한다.
- 닉네임은 한글, 영문, 숫자만 허용한다.

## 6. 정확도 계산

충전기 중심과 콘센트 중심 거리 기준으로 정확도를 계산한다.

### 예시 공식

```text
정확도 = 100 - (거리 × 보정값)
```

- 정확도는 최소 0%, 최대 100%로 제한한다.
- 리더보드에는 계산이 완료된 최종 정확도만 저장한다.

## 7. 판정 기준

| 정확도 | 판정 |
| --- | --- |
| 100% | PERFECT CHARGE |
| 90~99% | GREAT |
| 70~89% | GOOD |
| 40~69% | WEAK |
| 1~39% | BAD |
| 0% | MISS |

## 8. 결과 화면

### 출력 요소

- 정확도 %
- 판정 문구
- 배터리 충전 결과
- 기록 등록 버튼
- 리더보드 미리보기
- 다시하기 버튼

### 예시

```text
정확도 96%
GREAT CHARGE!
배터리 96% 충전됨
현재 순위 3위
```

## 9. 리더보드 화면

### 출력 요소

- 상위 10개 기록
- 내 최근 기록의 순위
- 닉네임
- 정확도
- 판정 문구
- 기록 시간

### 정렬 기준

1. 정확도 높은 순
2. 등록 시간 빠른 순

### MVP 저장 방식

- MVP에서는 서버 API를 통해 기록을 저장하고 조회한다.
- 이메일 로그인 계정 기반으로 기록한다.
- 리더보드에는 이메일을 노출하지 않고 닉네임만 노출한다.
- 닉네임은 2~12자로 제한한다.
- 닉네임은 한글, 영문, 숫자만 허용한다.
- 욕설 필터링은 MVP에서는 금칙어 목록 기반으로 처리한다.

## 10. 사운드

### MVP 필수 사운드

- 정확도 계산

- 이메일 회원가입

1. 충전기 이동과 타이밍 입력

## 14. MVP 완성 기준

아래 조건을 만족하면 MVP 완성으로 판단한다.

- 게임 시작 가능
- 충전기 자동 이동
- 입력 시 충전기 정지
- 정확도 출력
- 판정 문구 출력
- 회원가입 가능
- 로그인 가능
- 로그인된 유저의 플레이 결과 저장 가능
- 리더보드 조회 가능
- 반복 플레이 가능

## 15. 이메일 인증 추가 정책

- 회원가입 후 이메일 인증 메일을 발송한다.
- 이메일 인증을 완료해야 로그인할 수 있다.
- 인증 메일은 재발송할 수 있다.
- 이메일 인증 토큰은 만료 시간이 있어야 한다.
