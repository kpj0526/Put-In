# 충전기 타이밍 게임 Frontend

React, JavaScript, Vite 기반 프론트엔드입니다.

## 주요 화면

- 로그인/회원가입
- 시작 화면
- 게임 화면
- 결과 화면
- 리더보드 화면

## 환경 변수

`.env.example`을 참고해 `.env`를 생성합니다.

```bash
cp .env.example .env
```

기본 API 주소:

```env
VITE_API_BASE_URL=http://localhost:3000/api
```

## 실행

```bash
pnpm install
pnpm run dev
```

기본 개발 서버:

```text
http://localhost:5173
```

## 빌드

```bash
pnpm run build
```
